# 企业报告优化专家

专业的企业报告质量优化技能，自动检查报告质量并直接改进，补充缺失数据、深化分析、增加图表，确保报告达到银行审批标准（9分以上）。

## 功能特点

- **5大维度质量检查**：格式规范性、内容完整性、图表数量、分析深度、决策支持度
- **自动改进**：不是只生成质检报告，而是直接改进报告
- **评分机制**：1-10分量化评分，从6-7分提升到9.0-9.5分
- **改进优先级**：高/中/低优先级清单，确保关键问题优先解决

## 使用方法

```
使用 corporate-report-optimizer 技能优化【某某公司】的尽调报告
```

## 输出

- `尽调报告_{company_name}_{date}_改进版.docx`：改进后的报告（评分9.0-9.5分）
- `改进说明.md`：改进内容说明

## 前提条件

需要已有初版尽调报告。

## 独立使用

本技能可以独立使用，也可以作为完整尽调流程的一部分。

在 Manus 技能广场搜索 "corporate-report-optimizer" 并安装。
